As of June 23rd, 2026, only Andrew is contributing to this repo, so the AI policy is whatever I (Andrew) feel like.

I believe that AI systems will soon (1-4 years, approximately) be able to create software projects with the end-to-end quality level of an extremely good software engineer.
However, I've also found that current systems have deficiencies that are impossible to ignore. Code written by AI tends to overly focus on an explicit goal rather than the
overall health of the project and the documentation AI systems write is basically unuseable. Furthermore without any spot checks AI code has a tendency to fall into
whatever the current path is, which can include putting tens of thousands of lines of code into a single file.

Nevertheless, I find using AI coding agents makes me much more productive than I would be otherwise. They have nearly infinite endurance and can often solve problems
that I cannot just due to their tenacity and broad base of knowledge. Because of this I will be continuing to use them as I see fit.
